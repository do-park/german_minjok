## 목표

배달의 민족과 같은 음식 배달 플랫폼(소비자, 판매자)을 반응형 웹 형식으로 개발

## 기간

2020.05.18 ~ 2020.06.07

## 개발 환경

- Google Chrome Browser
- Python 3.7+
- Django 2.1.15
