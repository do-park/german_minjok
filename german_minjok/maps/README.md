## 작업하면서 느낀점

스크립트끼리 충돌로 지도안에서 검색이 되게끔 만들지는 못했다.

원래 예제에서는 다 나오던데 뭐가 문제인지는 좀더 파악 해 볼 필요가 있다.

더미데이터를 혼자 구현했는데 마스터로 옮기는데 문제가 다소 발생해서 초판본만 옮겼다.

넘기는 부분을 상의해서 좀더 다듬을 필요가 있음을 느꼈다.
